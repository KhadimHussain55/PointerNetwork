# Pointer network in PyTorch

This is an implementation of [Pointer Network](https://arxiv.org/abs/1506.03134) in PyTorch for [Convex Hull](https://en.wikibooks.org/wiki/Convexity/The_convex_hull) problem

## Environment
* Python 3.*
* Pytorch 0.3.*
* TensorboardX 1.1

## Data
